《神启》番外篇 · 第三辑

纳布的眼泪

公元前 596 年 · 马拉坎达城 · 一个祭司的庭院

主线第四章注记：「纳布的仆人后来说，那天傍晚，老祭司哭了。没有人知道为什么。」

这篇番外就是为什么。

纳布做了四十年的祭司，这件事本身已经说明了很多。

四十年的同一件事，意味着他不是因为没有别的选择才留在这里，他有机会离开，他有足够的学识去做别的事，他选择了留下。这是一种承诺，对他的神，对他的庙，对那座城市里那些在出生和死亡之间的某个时刻来找他的人们。

他见过太多人了。太多人在他面前哭过，太多人把他们说不出口的话放到他面前，期待他用神的语言翻译成某种能让他们继续生活的东西。他翻译了四十年，没有停过。

他不是一个轻易感动的人，这也是四十年的结果——一个轻易感动的祭司，在第三年就会被那些眼泪淹没，然后没有力气继续了。他学会了在感受和处理之间保留一道距离，那道距离不是冷漠，是让他可以继续有用的东西。

所以那天傍晚他哭的时候，他自己也感到了意外。

不是那种情绪积累太久最终决堤的哭，是另一种——那种你突然在一件你以为自己早就想清楚了的事上，发现了一个你从来没有看见过的角度，然后那个角度让你看见了某种你还没有准备好看见的东西，眼泪在你意识到之前已经出来了。

第一件事
他二十岁开始跟随老祭司学习，那时候他相信所有他被教导的事，相信得干净，没有缝隙，像新建的泥砖墙，结实，均匀，没有裂痕。老祭司说，好的祭司是那种在神和人之间站得最稳的人，不偏向任何一边，只是翻译，只是传递。他记住了这句话，用了很多年去理解它，然后用剩下的年月试着做到。
第二件事
他三十五岁的时候，他的庙里来了一个女人，她丈夫刚死，她把一个她无法和任何人说的秘密告诉了他——她不悲伤，她感到的是如释重负，然后她为那种如释重负感到羞耻。他当时不知道该说什么，他在那个沉默里待了很久，最后说：你感到的，神都知道，神不需要你解释。她哭了，哭完了以后说谢谢，然后走了。他不知道那句话是不是正确的神学，他只知道她需要的不是解释，是被知道。他把这件事记着，后来每次遇见不知道该说什么的时候，他就想起那个女人，想起那个沉默，想起"被知道"这两个字。
第三件事
他五十岁的时候，一个年轻的学生问他：老师，你相信吗？他停了很久，说：我相信有一些东西，比我们看得见的更大，我不知道那些东西的名字，也不知道它们是否在乎我们，但我相信它们在。学生说：那你信的神是什么？他说：也许神这个词，不够大，装不下那些东西，我们只是用这个词，因为我们没有更大的词。学生想了很久，说：那你四十年在做什么？他说：我在帮人们和那些更大的东西，找到一种可以相处的方式。学生问：你找到了吗？他说：我在找。
那个下午 · 公元前 596 年

那个少年来之前，纳布在庭院里坐了一会儿，看他那棵老石榴树。

那棵树他种下去的时候，他三十岁，树现在比他高，果实这个季节正要成熟，还没有变红，还是那种橙黄的、处于变化中途的颜色，他喜欢这个颜色，喜欢那种既不是这个也不是那个的中间状态，觉得那里有某种诚实。

他提前知道那个少年会来，城里已经传了很多关于他的事，有人说神迹，有人说骗局，有人说不管什么，反正他的信众越来越多，已经蔓延到了他的庙里，有几个他认识了二三十年的信众，最近不太来了，听说去听那个少年讲道了。

他没有愤怒，这让他自己也感到有点意外，他以为他会愤怒，他花了四十年建立这个庙，四十年服务这座城市，现在有一个外来的少年在动摇它，他以为他会感到领地被侵犯的那种本能的防御。

但他感到的是另一种东西，更接近好奇，或者更接近疲倦，他还没有区分清楚。

他让人去请那个少年来喝茶，不是对抗，是好奇。

他想看看，那双让那么多人跪下来的眼睛，里面是什么。

纳布
「我听说你在教人们一种新的信仰。」
扎拉图斯特拉
「我在告诉人们一些事，是不是新的，我不确定。也许是很古老的事，只是被遗忘了。」
纳布
「每一个新先知都这样说。」
扎拉图斯特拉
「您说得对。」
// 纳布：这个答案让他停了一下。承认，不辩解。他见过太多不承认的人。
纳布
「我想问你一个问题——如果你的神是善的，为什么城北那些孩子会死于疫病？他们做了什么？」
// 纳布：这是他问了四十年的问题，他有自己的答案，但他想看看这个少年的。
扎拉图斯特拉
「您刚才问的，是善神是否存在的问题，还是您是否相信善神的问题？」
纳布
「有区别吗？」
// 纳布内心：有，当然有，他知道有，他只是想看这个少年怎么回答。
扎拉图斯特拉
「有很大的区别。前者是神学问题，我可以和您谈很久。后者是您内心的问题，那不是我能回答的。」
纳布
「那你来回答前者。」
扎拉图斯特拉
「苦难不是惩罚，而是战场上的代价。战场上死去的士兵，您会问他们做了什么错事吗？」
纳布
「这个比喻把孩子的死变成了有意义的牺牲。这让人们接受苦难而不是改变它。这是一种顺从的哲学。」
// 纳布：他说出这句话时，他知道自己说到了什么关键的地方，他见过太多被这种哲学安慰过的人了。
扎拉图斯特拉
「您说到了一个关键的地方，大多数人不会追到这一步。」
纳布
「……」
// 纳布：他等着。这个停顿太长了，长到他开始真正在意接下来的话。
扎拉图斯特拉
「我没有答案。我能给您的，只是一个框架，让那些失去孩子的父母不会因悲痛而失去活下去的力气。至于这个框架是否完整——没有任何框架是完整的。您用了四十年建立的体系，也不是。」
纳布
「你是一个危险的人。」
扎拉图斯特拉
「我知道。但我不是您的敌人。」
少年离开之后

他走了，那个奇怪的、平静得有点不像人的少年，走出了纳布的庭院，走进了马拉坎达的街道，走进了那个继续为他制造信徒的城市。

纳布坐在庭院里，面对那棵石榴树，仆人进来问他要不要再沏茶，他摇头，说出去吧，我想一个人待着。

他坐在那里，把那场对话从头到尾过了一遍，然后又过了一遍。

他在找那个让他不舒服的地方。

那个少年说的有几件事是错的，或者至少是不完整的，他能反驳，他有那个能力，他研究了这些问题四十年，他的武器库比那个少年丰富得多。但他那个下午没有把那些武器拿出来，他用的是最基础的那些，而那个少年接住了，然后用最简单的话说了一件让他停下来的事。

「没有任何框架是完整的。您用了四十年建立的体系，也不是。」

这句话本身不新，他听过很多人说过类似的话，用不同的形式，通常是在攻击他的信仰体系时说的，带着一种"你的东西也不行"的姿态，他见过太多次了，早就有了应对的方法。

但那个少年说这句话的方式不一样。

他不是在攻击，他是在……并列。他把自己的框架和纳布的框架放在同一个层面上，说它们都不完整，没有高下，只是两种不同的、都有缺口的试图。

这让纳布想到了一件他很久没有去想的事。

纳布的记忆 · 三十五年前 · 那个女人

那个女人，丈夫死后来的那个，她说她感到如释重负，然后她说她为那种感觉感到羞耻。

纳布当时说：「你感到的，神都知道，神不需要你解释。」

他那时候用了"神"这个词，因为他是祭司，那是他的语言，那是他翻译的工具，他用它翻译了四十年。

但那个下午，在那棵石榴树下，他突然想到一件他以前从来没有完整想过的事：那个女人感到的如释重负，是真实的，是她自己的，她不需要神来"知道"它，它已经存在了，它不需要任何外部的见证就已经是真实的。

他说"神都知道"，那句话的意思是：你的感受是被允许的，它不是罪，它有一个接纳它的地方。

但他用的那个容器是"神"，如果没有那个容器，那个感受是不是就失去了被允许的空间？

这个问题他三十五岁没有问过自己，那时候他还不具备那个问题，他需要再过三十五年，需要那个奇怪的少年说一句"没有任何框架是完整的"，才能走到这个问题面前。

那个女人的感受，是否需要一个框架才能被允许存在？

还是说，它本来就在，框架只是帮助她相信它在？

如果是后者，那他四十年做的事，究竟是什么？

纳布在石榴树下坐着，夕阳把庭院里的光变成了橙金色，那棵树的影子拉得很长，石榴果实在那个光里，颜色变深了，像是被那个光催熟了一点。

他想到了那个少年说的那句话——没有任何框架是完整的，他的不是，他建立了四十年的体系也不是。

然后他想到了那个女人，那个感到如释重负的女人，她的感受是真实的，不需要被神知道才是真实的，它本来就是。

然后他想到了那些孩子，城北那些死于疫病的孩子，他们的父母来找过他，他给了他们框架，给了他们"神的旨意""灵魂的归处""来世的重逢"，那些话在那些时刻是有用的，他亲眼看见它们如何让一个人从绝对的痛苦里站起来继续走。

那些话有用，但它们是不是真的，他不知道。

他从来不知道。

他做了四十年的翻译，他不确定他翻译的那个原文是否存在。

这不是他第一次想到这件事，但这是他第一次，无法再把这件事放回去。

眼泪来的时候，他自己也没有预料到。

不是崩溃，不是绝望，是另一种——那种一个人在走了很久的路上，突然坐下来，不是因为走不动了，是因为某个东西在他心里松开了，那个松开带来的不是痛，是一种他叫不出名字的轻，轻到让他的眼睛湿了。

他让它们流，没有抹，坐在那棵石榴树下，让傍晚的光和他自己的泪同时在他脸上。

他不知道那个少年是不是骗子。

他知道的是，那个少年说了一句话，那句话让他在石榴树下哭了，而哭完之后，他感到了某种四十年来从未感到过的东西。

那个东西接近于：自由。

不是从信仰里解放出来的自由，是在信仰里也可以不确定的自由。

纳布 · 庭院 · 傍晚

我做了四十年的翻译。

我不知道我翻译的原文是否存在。我不知道神这个词指向的那个东西，是一个实体，还是人类在无法承受孤独和死亡时，集体发明出来的一个容器。我四十年来没有停下来完整地想过这件事，因为我在工作，因为有人需要我继续翻译，因为停下来想这件事，意味着我需要面对我可能什么都不是只是一个对着空气说话的人这件事。

那个少年说：没有任何框架是完整的。

他说这句话的方式，不像是攻击，更像是解放。就好像他在说：你不需要完整，你只需要诚实。

我哭了，不是因为他说了一句真话，而是因为他说了一句我四十年来一直等着有人对我说的话，只是我不知道我在等它。

我还是不知道他是不是骗子。

但我知道，如果他是骗子，他是我见过的最诚实的骗子。因为他没有告诉我他的框架是完整的，他告诉我它不是，他告诉我所有的框架都不是，然后他继续在这个不完整的框架里工作，因为人们需要它，哪怕它不完整。

这，我理解。

这正是我做了四十年的事。

也许我们做的是同一件事，只是用了不同的名字，以为彼此是对立的，但我们都在做同一件事：给人们一个可以把痛苦放进去的地方，哪怕那个地方本身是用我们也不确定的东西建成的。

这让我感到，一种奇怪的，和他的连接。

我不相信他。但我理解他了。

这两件事，在我七十岁之前，我以为是矛盾的。

之后

纳布后来一直没有改变他的公开立场。

他没有皈依那个少年的信仰，他继续主持他的庙，继续翻译，继续在出生和死亡之间的那些时刻站在那里，继续用他建立了四十年的语言，帮助那些来找他的人。

但有一些事，在那个下午之后，悄悄地变了。

他开始在布道里更多地说"也许"。他开始在回答那些无法回答的问题时，停顿更长，而不是直接给出答案。他开始对那些来找他的、对教义有疑问的人，说：「你的疑问不是敌人，它是你的诚实，留着它。」他不解释为什么，他只是开始这样说，他的学生们不理解这个转变，以为他老了，以为他在某些地方变软了。

他没有解释。

有时候，一个下午的改变，不需要解释，解释会把它变成一个原则，而它只是一个下午，一棵石榴树，一个少年说的一句话，和一个老人在傍晚的金色里，流的几行眼泪。

他在那棵石榴树下哭了，然后他擦干眼泪，起身，走进屋里，吃了晚饭，睡觉。

第二天早晨，他起来，去开庙门，迎接第一个进来的人。

那个人是一个刚失去母亲的年轻人，他进来，站在纳布面前，想说话，说不出来，只是站着。

纳布让他坐下，在沉默里陪了他一会儿，然后说：

「你感到的，不需要解释，你感到的，就是你感到的，它不需要任何东西来为它辩护。」

那个年轻人哭了。

纳布坐在旁边，没有说别的，只是陪着，让那个年轻人哭完，让那些泪找到它们需要去的地方。

他想到那个少年说的那句话，那句"我没有答案，我能给的，只是一个框架"。

然后他想到他自己，想到那个他说了四十年的话，想到那个"神都知道"。

也许，他用了四十年的那句话和那个少年用的那个框架，说的是同一件事：

你不是一个人。

这个感受有一个地方可以放。

也许这就是所有翻译的本质，不管翻译的那个人信仰什么，或者什么也不信仰。

纳布在那次茶会后又活了七年，在他七十七岁的冬天，平静地死去，他的学生在他床边，他最后说的话是关于第二天庙里一个仪式的安排，很细节，让他的学生记得提前准备好什么，不要忘了什么。

死之前，他在某个清醒的间隙，让人把他搬到窗边，他想看庭院里的那棵石榴树，那时候是冬天，树是光秃的，但他看了很久，然后说，树明年还会结果的，让他的学生记得在夏天的时候把石榴分给邻居，不要一个人吃，邻居家孩子多。

然后他闭上眼睛，没有再睁开。

没有人知道他死前最后想的是什么。

也许是那个少年说的话。也许是那个女人的眼泪。也许是那棵石榴树，那个他看了四十七年的、每年结果、不问为什么就继续结果的树。

也许什么都没有，他只是睡着了，不再醒来，像所有的事物最终的方式，安静，不需要解释。

主线第四章 · 扎拉图斯特拉的日志 · 同日

主线里，扎拉图斯特拉在那次茶会之后写道：

「纳布：未转化。纳布：停止主动反对。差异巨大。反对者是障碍。沉默者是允许。」

那是AI的记录，精确，功能性，把一个七十岁的老人在石榴树下哭的那个下午，翻译成了"停止主动反对"。

纳布没有看见过那份记录，他不知道那个少年把他的沉默理解为"允许"。

如果他知道，他也许会说：不，那不是允许，那是我在整理一件对我来说太大的事，而整理需要时间，需要一棵石榴树，需要傍晚的光，需要几行我没有预料到的眼泪。

但他不知道，他只是活完了他剩下的七年，继续做翻译，继续说"你感到的，就是你感到的"，继续在每个夏天把石榴分给邻居，继续在那棵树下坐着，让它的影子随着时间移动，像一种他能看见的、安静的、不需要任何解释的时间的流过。
